## 一、應收帳款帳齡分析

### 功能概述

在儀表板（首頁）顯示未收款的帳齡分布，依發票預計收款日將未收金額分類，便於催收與現金流管理。

### 顯示位置

儀表板 → 「已開立發票未收款總額」卡片下方

### 帳齡區間

| 區間 | 說明 |
|------|------|
| 未到期 | 預計收款日尚未到期 |
| 1-30 天 | 逾期 1～30 天 |
| 31-60 天 | 逾期 31～60 天 |
| 61-90 天 | 逾期 61～90 天 |
| 90 天以上 | 逾期超過 90 天 |
| 未設預計日 | 發票未設定預計收款日 |

### 計算邏輯

- 僅統計**有效發票**（status = 有效）
- 各發票未收金額 = 發票金額 − 已收款（含匯費差異）
- 帳齡依 `expected_payment_date` 或 `invoice_date` 計算
- 支援儀表板年度篩選

### 權限

僅 **admin**、**user** 角色可見；**salesperson**、**boss** 不顯示。

### 相關檔案

- `src/services/ReceivablesAgingService.js` - 帳齡計算服務
- `src/routes/index.js` - 傳遞帳齡資料至儀表板
- `src/views/index.ejs` - 帳齡區塊 UI

## 二、全域快速搜尋

### 功能概述

支援跨專案、客戶、發票的關鍵字搜尋，加快日常查詢效率。

### 入口

- 導覽列「搜尋」連結
- 導覽列右側搜尋框（需登入）

### 搜尋範圍

| 類別 | 搜尋欄位 |
|------|----------|
| 專案 | 專案編號、專案名稱 |
| 客戶 | 客戶編號、公司名稱 |
| 發票 | 發票號碼 |

### 使用方式

1. 輸入至少 2 個字元
2. 按 Enter 或點擊搜尋按鈕
3. 結果分類顯示：專案、客戶、發票
4. 每類最多 20 筆

### 權限

- 依使用者角色過濾專案（業務員僅能搜尋自己的專案）
- 客戶與發票依專案權限顯示

### 相關檔案

- `src/routes/search.js` - 搜尋路由
- `src/views/search/index.ejs` - 搜尋結果頁
- `src/views/layout.ejs` - 導覽列搜尋框
- `src/app.js` - 註冊 `/search` 路由
