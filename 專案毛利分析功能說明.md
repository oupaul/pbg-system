# 專案毛利分析功能說明

## 更新日期
2026-02-07

---

## 一、功能概述

專案毛利分析依「方案 A」計算，以專案未稅價格作為收入，扣除專案成本後得出毛利與毛利率，便於檢視各專案、業務、類型的獲利狀況。支援毛利警示閾值、Excel 及 PDF 匯出。

---

## 二、計算公式

| 項目 | 說明 |
|------|------|
| 收入 | 專案未稅價格（`price_without_tax`） |
| 成本 | 專案成本總和（`costs` 表 `amount` 加總） |
| 毛利 | 收入 − 成本 |
| 毛利率 | 毛利 ÷ 收入 × 100% |

---

## 三、顯示位置

導覽列 → **毛利分析**（僅 admin、user 可見）

---

## 四、報表內容

### 彙總卡片

- 總收入（未稅）
- 總成本
- 總毛利
- 平均毛利率

### 專案明細分頁

| 欄位 | 說明 |
|------|------|
| 專案編號 | 可點擊連結至專案詳情 |
| 專案名稱 | 專案說明 |
| 年度 | 簽約年度 |
| 類型 | 專案類型 |
| 業務 | 負責業務 |
| 收入（未稅） | 專案未稅價格 |
| 成本 | 專案成本總和 |
| 毛利 | 收入 − 成本 |
| 毛利率 | 百分比，低於該專案所屬類型閾值時顯示警示 |

### 依業務彙總分頁

依業務人員彙總專案數、總收入、總成本、總毛利、毛利率，可連結至該業務的專案列表。

### 依類型彙總分頁

依專案類型彙總專案數、總收入、總成本、總毛利、毛利率。

---

## 五、毛利警示閾值

- 可於 **專案類型管理** 為各類型設定「毛利警示閾值(%)」
- 警示針對**每個專案**：若該專案毛利率低於其所屬類型的閾值，於專案明細分頁顯示警示（黃底、紅字、警示 badge）
- 閾值不是針對類型彙總，而是針對單一專案與其類型對應的閾值比較

---

## 六、Excel 匯出

- 路徑：`GET /gross-profit/export?year=2024` 或 `?year=all`
- 檔名：`毛利分析_2024.xlsx` 或 `毛利分析_全部.xlsx`
- 三個工作表：**專案明細**、**依業務彙總**、**依類型彙總**
- 依目前頁面篩選的年度匯出

---

## 七、PDF 匯出

- 路徑：`GET /gross-profit/export/pdf?year=2024` 或 `?year=all`
- 檔名：`毛利分析_2024.pdf` 或 `毛利分析_全部.pdf`
- 內容：標題、彙總資訊（總收入、總成本、總毛利、平均毛利率）、專案明細表格（橫向 A4）
- 依目前頁面篩選的年度匯出

---

## 八、年度篩選

- **全部年度**：不限定專案簽約年度
- **指定年度**：僅統計該年度專案

---

## 九、權限

僅 **admin**、**user** 角色可檢視毛利分析頁面。

---

## 十、相關檔案

- `src/services/GrossProfitAnalysisService.js` - 毛利分析服務
- `src/services/ExcelExportService.js` - Excel 匯出（`exportGrossProfit`）
- `src/services/PdfExportService.js` - PDF 匯出（`exportGrossProfit`）
- `src/routes/grossProfit.js` - 路由（含 `/export`、`/export/pdf`）
- `src/views/gross-profit/index.ejs` - 視圖
- `src/app.js` - 路由註冊
- `src/views/layout.ejs` - 導覽連結

---

## 十一、資料來源

- **收入**：`projects.price_without_tax`
- **成本**：`costs` 表，每筆專案可有多筆成本記錄，由專案詳情頁「新增成本」維護
