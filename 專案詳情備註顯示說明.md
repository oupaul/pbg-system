# 專案詳情備註顯示說明

## 功能概述

專案的「備註」欄位（資料庫 `projects.notes`，經 `v_project_summary` 查詢）原本僅在新增／編輯專案、專案範本等表單中維護。自 **2026-03-27** 起，**專案詳情頁**（`/projects/:id`）的「專案資訊」卡片內會一併顯示備註，方便檢視與對照，無需進入編輯頁。

## 使用者介面

- **位置**：專案詳情 → 左側「專案資訊」卡片 → 「客戶」欄位下方 → **備註**。
- **有內容**：以淺底、圓角區塊顯示全文；**保留換行**（`white-space: pre-wrap`）。
- **無內容**：顯示「尚無」。
- **編輯**：仍於「編輯專案」表單中修改備註；詳情頁為唯讀顯示。

## 技術實作

| 項目 | 說明 |
|------|------|
| 資料來源 | `Project.findById` → `v_project_summary`，欄位 `notes` 與編輯表單相同 |
| 視圖 | `src/views/projects/show.ejs`，於專案資訊區塊新增備註列 |
| 安全性 | 備註輸出前進行 HTML 跳脫（`&`, `<`, `>`, `"`），避免 XSS |

## 相關檔案

- `src/views/projects/show.ejs` — 專案詳情畫面
- `src/models/Project.js` — `findById`（沿用既有查詢）
- `src/routes/projects.js` — `GET /projects/:id` 傳入 `project`（無需額外欄位）

## 測試建議

1. 編輯某專案並填入多行備註，儲存後開啟該專案詳情，確認文字與換行正確。
2. 清空備註或新建無備註專案，詳情頁應顯示「尚無」。
