## 📦 快速複製指令（Linux/Mac）

### 方法 1: 使用 rsync（推薦）
```bash
rsync -av --exclude='node_modules' \
          --exclude='.git' \
          --exclude='*.log' \
          --exclude='data' \
          --exclude='uploads' \
          --exclude='*.tar.gz' \
          --exclude='*.xlsx' \
          --exclude='*.bat' \
          --exclude='*.ps1' \
          /來源路徑/專案發票獎金計算系統/ \
          新主機用戶@新主機IP:/目標路徑/
```

### 方法 2: 使用 tar 打包傳輸
```bash
# 在本地打包
tar -czf deploy.tar.gz \
  --exclude='node_modules' \
  --exclude='.git' \
  --exclude='data' \
  --exclude='uploads' \
  --exclude='*.log' \
  --exclude='*.tar.gz' \
  --exclude='*.xlsx' \
  src/ public/ migrations/ \
  deploy.sh backup.sh restore.sh uninstall.sh setup-backup-timer.sh \
  package.json package-lock.json

# 傳輸到新主機
scp deploy.tar.gz 新主機用戶@新主機IP:/目標路徑/

# 在新主機上解壓
cd /目標路徑
tar -xzf deploy.tar.gz
```

### 方法 3: 使用 SCP（簡單複製）
```bash
# 複製整個目錄（會包含 node_modules，但可以在新主機上刪除）
scp -r /來源路徑/專案發票獎金計算系統/ 新主機用戶@新主機IP:/目標路徑/

# 在新主機上刪除不需要的檔案
cd /目標路徑/專案發票獎金計算系統
rm -rf node_modules .git data/uploads/* *.log *.tar.gz *.xlsx
```

## 🚀 部署步驟

1. **複製檔案**（使用上述任一方法）

2. **在新主機上設定權限**
```bash
cd /目標路徑/專案發票獎金計算系統
chmod +x deploy.sh backup.sh restore.sh uninstall.sh setup-backup-timer.sh
```

3. **執行部署**
```bash
sudo ./deploy.sh
```

部署腳本會自動：
- 檢查並安裝 Node.js（如需要）
- 安裝 npm 依賴套件
- 執行資料庫遷移
- 建立必要的目錄（data/, uploads/）
- 設定 systemd 服務
