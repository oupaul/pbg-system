# v1.14.0 功能更新說明

**更新日期**：2026-02-06

## 更新摘要

本版本新增業務績效圖表、收款提醒、發票／收款軟刪除、部分折讓、專案範本、稽核紀錄匯出，並改進發票「更多」下拉選單的版面顯示。

---

## 一、業務績效儀表板圖表

- **圓餅圖**：各業務「已開發票」占比（使用 Chart.js 4.4.1）
- **長條圖**：各業務「總獎金 / 已發放 / 待發放」三組長條
- 僅在有資料時顯示圖表
- **相關檔案**：`src/views/sales-performance/index.ejs`、`src/services/SalesPerformanceService.js`

---

## 二、發票／收款提醒（站內提醒）

- 依「預計收款日」在首頁顯示站內提醒，不需 Email
- **即將到期**：到期日在今天～今天+N 天內
- **已逾期**：已過預計收款日
- 天數由系統設定 `payment_reminder_days` 控制（預設 7 天）
- **相關檔案**：`src/services/ReceivablesAgingService.js`（`getPaymentReminder`）、`src/routes/index.js`、`src/views/index.ejs`

---

## 三、發票與收款軟刪除

- 發票、收款改為軟刪除（設定 `deleted_at`），不直接從資料庫刪除
- **還原**：專案詳情頁可「顯示已刪除」，並對已刪除項目執行「還原」
- **遷移**：`npm run migrate:soft-delete` 或 `node migrations/migrate_soft_delete_invoices_payments.js`
- **相關檔案**：`migrations/migrate_soft_delete_invoices_payments.js`、`src/models/Invoice.js`、`src/models/Payment.js`、`src/routes/invoices.js`、`src/routes/payments.js`、`src/routes/projects.js`、`src/views/projects/show.ejs`

---

## 四、發票部分折讓

- 發票可設定「部分折讓」：認列金額 = 原金額 − 折讓金額
- 發票仍為有效，僅認列金額變更
- **遷移**：`npm run migrate:partial-allowance` 或 `node migrations/migrate_invoice_partial_allowance.js`
- **相關檔案**：`migrations/migrate_invoice_partial_allowance.js`、`src/models/Invoice.js`、`src/routes/invoices.js`、`src/views/projects/show.ejs`

---

## 五、專案範本

- 新增「專案範本」管理：建立、編輯、刪除範本
- 範本欄位：範本名稱、說明、專案類型、業務、客戶、專案名稱、價格、銷貨折讓、備註等
- **從範本建立專案**：`/projects/new?from_template=:id` 會帶入範本欄位
- **遷移**：`npm run migrate:project-templates` 或 `node migrations/migrate_project_templates.js`
- **相關檔案**：`migrations/migrate_project_templates.js`、`src/models/ProjectTemplate.js`、`src/routes/projectTemplates.js`、`src/views/project-templates/`、`src/views/layout.ejs`（導覽列新增「專案範本」）

---

## 六、稽核紀錄匯出

- 修改記錄頁新增「匯出 CSV」按鈕
- 依目前篩選條件匯出，最多 10000 筆
- 格式：UTF-8 BOM，欄位為時間、操作、資料表、記錄ID、使用者、舊值、新值
- 檔名：`audit-logs-YYYY-MM-DD-HHmm.csv`
- **相關檔案**：`src/routes/auditLogs.js`、`src/views/audit-logs/index.ejs`

---

## 七、發票「更多」下拉選單改進

- 選單改為向下展開（不再向左展開）
- 使用 `overflow: visible` 避免被「收款明細」等區塊裁切
- 加大選項內距與高度，提升閱讀與點選
- **相關檔案**：`src/views/projects/show.ejs`、`public/css/style.css`

---

## 部署時需執行的遷移

```bash
npm run migrate:soft-delete      # 若尚未執行
npm run migrate:partial-allowance
npm run migrate:project-templates
```
