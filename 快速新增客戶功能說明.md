# 快速新增客戶功能說明

## 📋 功能概述

在新增或編輯專案時，如果找不到需要的客戶，可以直接點擊「快速新增」連結，在彈出視窗中快速新增客戶，新增成功後會自動選擇該客戶，無需離開當前頁面。

## 🎯 使用場景

- **新增專案時發現客戶不存在**：無需跳轉到客戶管理頁面
- **提升工作效率**：一次操作完成客戶新增和選擇
- **減少頁面跳轉**：保持當前表單的填寫狀態

## 💡 使用方法

### 方式 1：使用「快速新增」連結

1. **進入專案表單**：點擊「新增專案」或「編輯專案」
2. **找到客戶欄位**：在表單中找到「客戶」選擇框
3. **點擊快速新增**：點擊欄位下方的「快速新增」連結
4. **填寫客戶資料**：
   - 客戶編號（必填）
   - 公司名稱（必填）
   - 統一編號（選填，8 位數字）
   - 是否新客戶（選填）
5. **提交**：點擊「新增並選擇」按鈕
6. **自動選擇**：新增成功後，客戶會自動被選擇

### 方式 2：Select2 搜尋後新增

1. **使用 Select2 搜尋**：在客戶欄位輸入關鍵字
2. **找不到時**：如果找不到符合的客戶
3. **點擊快速新增**：點擊「找不到客戶？快速新增」連結
4. **其餘步驟同方式 1**

## 📊 技術實現

### 後端 API

#### 新增路由：`POST /customers/quick-add`

**檔案**：`src/routes/customers.js`

**請求格式**：
```json
{
  "customer_code": "C001",
  "company_name": "測試公司",
  "tax_id": "12345678",
  "is_new_customer": "1"
}
```

**成功回應**：
```json
{
  "success": true,
  "customer": {
    "id": 123,
    "customer_code": "C001",
    "company_name": "測試公司",
    "tax_id": "12345678"
  }
}
```

**失敗回應**：
```json
{
  "success": false,
  "error": "錯誤訊息"
}
```

## 🔍 資料驗證

### 必填欄位

- ✅ **客戶編號**：不可為空
- ✅ **公司名稱**：不可為空

### 選填欄位

- ⭕ **統一編號**：8 位數字（選填）
- ⭕ **新客戶**：勾選框（預設不勾選）

### 格式驗證

- **統一編號**：HTML5 pattern 驗證 `[0-9]{8}`
- **客戶編號**：無特殊格式要求
- **公司名稱**：無特殊格式要求

## 🐛 錯誤處理

### 常見錯誤

#### 1. 客戶編號重複

**錯誤訊息**：「客戶編號已存在」

**解決方案**：
- 使用不同的客戶編號
- 或在客戶管理中查找現有客戶

#### 2. 必填欄位未填

**錯誤訊息**：「請填寫客戶編號和公司名稱」

**解決方案**：
- 確認已填寫客戶編號
- 確認已填寫公司名稱

#### 3. 統一編號格式錯誤

**錯誤訊息**：「請輸入有效的統一編號（8 位數字）」

**解決方案**：
- 確認統一編號為 8 位數字
- 或留空（選填欄位）

#### 4. 網路錯誤

**錯誤訊息**：「新增客戶時發生錯誤：...」

**解決方案**：
- 檢查網路連線
- 確認伺服器運行正常
- 重試操作
